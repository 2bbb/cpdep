# cpdep

analyse & copy dependencies on C++ library

## install

```
npm install -g 2bbb/cpdep
```

## how to use

```
cpdep --help
```

```
cpdep -c -d ./boost_flat_map -r ./booooost/boost_1_58_0 boost/container/flat_map.hpp
```
